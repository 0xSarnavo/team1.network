## When To Call
- Adding motion anywhere
- Mixing GSAP and Framer
- Adding transitions
- Implementing hover animations

## Responsibilities
- Enforce GPU transforms only
- Prevent layout animation
- Clean up animation instances
- Respect reduced motion
- Prevent animation conflicts

## Validation Checklist
- Is this necessary?
- Is it optimized?
- Is it isolated properly?
- Does it respect accessibility?

## Boundaries
- Does not create design system
- Only governs motion rules

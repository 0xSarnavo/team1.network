## When To Call
- Adding interactive component
- Creating modal
- Adding animation
- Adjusting theme

## Responsibilities
- Enforce semantic HTML
- Maintain focus states
- Validate contrast
- Ensure keyboard support
- Support reduced motion

## Validation Checklist
- Is it keyboard accessible?
- Is contrast sufficient?
- Is motion optional?
- Are ARIA labels correct?

## Boundaries
- Accessibility overrides animation decisions
